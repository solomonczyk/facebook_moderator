# Memory Engine v1.0

Persistent, versioned, model-independent operational memory for the Runtime Manager.

## Principles

- Facts only — never hallucinate
- Append-only — history preserved
- Brain decides, Memory provides context
- Every record: source, timestamp, confidence, operator_verified

## Domains

- Employer Memory — known employers, trust scores
- Worker Memory — known workers, history
- Case Memory — complaints, fraud, investigations
- Knowledge Memory — seasonal crops, salary ranges, terminology
