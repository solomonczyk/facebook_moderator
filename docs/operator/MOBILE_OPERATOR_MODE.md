# Mobile Operator Mode — Sezonski rad Srbija

> Kako voditi grupu sa telefona, 10 minuta dnevno.

## Komande za mobilni režim

| Komanda | Šta radi |
|---------|---------|
| `/fb_post <tekst>` | Kopirani FB post — klasifikuje i sprema odgovor |
| `/fb_comment <tekst>` | Kopirani FB komentar |
| `/fb_question <tekst>` | Pitanje člana |
| `/fb_review <tekst>` | Iskustvo / recenzija |
| `/fb_member <tekst>` | Profil člana |
| `/capture <tekst>` | Post iz druge grupe (za digest) |
| `/reply <tekst>` | Analiza poruke + predlog odgovora |
| `/today` | Današnji pregled |
| `/links` | Forme za copy-paste |
| `/evening` | Večernji pregled po prioritetu |
| `/digest_next` | Digest za sutra |

## Kako kopirati post sa Facebook-a na telefonu

1. Otvorite Facebook aplikaciju
2. Nađite post/komentar
3. Pritisnite "Share" → "Copy" (ili dugo pritisnite tekst → Copy)
4. Otvorite Telegram → bot chat
5. Ukucajte `/fb_post ` i paste-ujte tekst

## 10 minuta uveče

1. `/evening` — pregled šta treba uraditi
2. `/drafts` — gotovi tekstovi za objavu
3. Kopirajte svaki draft → paste u Facebook grupu
4. `/digest` ili `/digest_next` — pripremite digest za sutra
5. Objavite digest ručno u grupu

## Šta NE raditi

- Ne objavljivati automatski — agent samo priprema tekst
- Ne koristiti reči "provereno", "sigurno", "garantovano"
- Ne objavljivati tuđe brojeve telefona bez dozvole
- Ne odgovarati na spam — obeležiti i ignoriše se

## Hitno zaustavljanje

Ako agent pošalje nešto pogrešno:
- `/status` — proveriti da li su svi gate-ovi DISABLED
- Bot ne objavljuje u Facebook — sve je ručno
