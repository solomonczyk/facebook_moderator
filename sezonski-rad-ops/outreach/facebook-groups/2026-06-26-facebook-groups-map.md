# Facebook Groups Outreach Map — 2026-06-26

## Important Note on Methodology

Facebook groups are NOT indexed by Google or any web search engine. They can only be found via Facebook's own internal search bar. This report provides:

- **Search queries** the operator pastes directly into Facebook search
- **Realistic group names** based on standard Serbian FB group naming conventions
- **Known groups** confirmed from earlier market research (news articles, 021.rs ads, Euronews reporting)
- **URL confirmation status** for every entry — be honest about what needs operator verification

Every group marked `url_confirmed: false` requires the operator to search Facebook directly before posting. This takes ~10 seconds per group.

## Posting Strategy

### Golden Rules

1. **Never spam.** One quality post per group. Not multiple posts. Not daily bumps.
2. **Read the rules first.** Every group. Before posting. Takes 30 seconds.
3. **If unclear — ask the admin.** A 2-sentence message: "Zdravo, adminiram grupu za sezonske radnike u Srbiji. Da li je u redu da podelim informaciju u vašoj grupi?"
4. **No fake engagement.** Don't ask friends to like/comment. Don't use fake accounts.
5. **Wait 24h between groups.** Don't post in 10 groups on the same day — Facebook flags this as spam.
6. **Track everything.** Update the outreach log after each action.
7. **If removed — accept it.** Don't argue with admins. Move on.

### Posting Cadence

| Day | Action |
|-----|--------|
| Day 1 | Post in 3 priority_1 groups (worker_help angle) |
| Day 2 | Post in 3 more priority_1 groups |
| Day 3 | Post in 3 priority_3 groups (community_support angle) |
| Day 4 | Review results. Check for comments/reactions. |
| Day 5 | Post in remaining priority_1 + selected priority_2 |
| Day 6+ | Monitor, respond to comments, track new members |

**Maximum: 3 posts per day. Never more.**
