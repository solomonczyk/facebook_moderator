# Ответ при личных данных

```
Ваша публикация не может быть одобрена, пока не будут удалены личные данные других людей.

В группе запрещено публиковать: JMBG, номера паспортов, домашние адреса, чужие номера телефонов без согласия, скриншоты личных переписок и фотографии документов.

Вы можете описать ситуацию без публикации личных данных. Когда вы отредактируете публикацию, мы сможем проверить её повторно.

Спасибо за понимание!
```
