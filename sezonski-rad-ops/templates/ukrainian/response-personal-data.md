# Відповідь при особистих даних

```
Ваша публікація не може бути схвалена, поки не будуть видалені особисті дані інших людей.

У групі заборонено публікувати: JMBG, номери паспортів, домашні адреси, чужі номери телефонів без згоди, скріншоти особистих переписок і фотографії документів.

Ви можете описати ситуацію без публікації особистих даних. Коли ви відредагуєте публікацію, ми зможемо перевірити її повторно.

Дякуємо за розуміння!
```
