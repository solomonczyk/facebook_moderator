# Răspuns la date personale

```
Publicația dvs. nu poate fi aprobată până când datele personale ale altor persoane nu sunt eliminate.

În grup este interzisă publicarea: JMBG, numere de pașaport, adrese de domiciliu, numere de telefon ale altor persoane fără consimțământ, capturi de ecran ale mesajelor private și fotografii ale documentelor.

Puteți descrie situația fără a publica date personale. După ce editați publicația, o putem verifica din nou.

Vă mulțumim pentru înțelegere!
```
