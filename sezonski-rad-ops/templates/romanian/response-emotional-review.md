# Răspuns la recenzie emoțională

```
Vă mulțumim că ați împărtășit experiența. Pentru a o putea publica, vă rugăm să scrieți mai concret și fără insulte:

- Unde și în ce perioadă ați lucrat
- Ce s-a promis (salariu, cazare, masă)
- Ce s-a întâmplat în realitate
- Dacă salariul a fost plătit

Este important să scrieți din experiență personală. Astfel ajutați alți lucrători și evitați problemele legale.

Vă mulțumim pentru înțelegere!
```
