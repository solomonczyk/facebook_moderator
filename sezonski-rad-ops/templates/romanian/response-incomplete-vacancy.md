# Răspuns la anunț incomplet

```
Vă mulțumim pentru anunț. Pentru a-l putea publica, vă rugăm să completați următoarele informații:

📍 Locul de muncă / orașul
🛠️ Tipul de muncă
📅 Când începe munca
👥 Câți lucrători sunt necesari
💰 Salariul
💳 Modul de plată
🕐 Programul de lucru
🏠 Cazare: da/nu
🍽️ Masă: da/nu
🚌 Transport: da/nu
📞 Telefon de contact / Viber / WhatsApp
✅ Sunt lucrătorii înregistrați oficial

După completare, anunțul poate fi aprobat. Vă mulțumim pentru înțelegere!
```
